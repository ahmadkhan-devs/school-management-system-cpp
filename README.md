# School Management System (C++)

This is a console-based School Management System developed in C++.  
The program uses file handling to store and manage student records.

## Features
- Add Student
- View All Students
- Search Student
- Remove Student
- Change Class
- Update Fee Status
- Show Fee Defaulters

## Technologies Used
- C++
- File Handling
- Console Interface

## How to Run
1. Download the repository
2. Compile the C++ file
3. Run the program
